Pose extraction

Layer 2 · d=2
Causal Conv1D with kernel size 3, dilation 2. Skips one L1 frame between each tap, expanding coverage cheaply.

GlobalAvgPool
Averages the 45 frame-level 32-D vectors into a single fixed-size 32-D embedding for the whole sequence.
45 × 32-D → 32-D
